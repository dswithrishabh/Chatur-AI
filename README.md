# Chatur AI

Chatur AI is an intelligent AI assistant built using **Streamlit**, **LangChain**, and **HuggingFace Transformers**.  
It can answer questions, provide predefined responses, and retrieve information from uploaded documents (PDF/TXT).

---

## Features

- Predefined custom replies (e.g., team info, greetings, motivational messages)
- RAG (Retrieval-Augmented Generation) for document-based answers
- Upload TXT and PDF files to improve AI knowledge
- Multi-language support (English + Hindi)
- User-friendly chat interface with chat history and rating system
